Module: shared_js/validation

Validation js sets up any custom validation by looking for relevant classes (see survey-docs) and also a MutationObserver to keep an eye on Qualtrics validation messages.

If it receives any error message from the validation functions or Qualtrics watch, fires addMessage

Bit more complicated than other modules, so see Validation for detailed explanation
